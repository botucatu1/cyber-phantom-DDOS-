## 📌 Description

CyberPhantom is an educational project developed to explore and understand network behavior, request patterns, and system responses under controlled conditions.

⚠️ This project should only be used in controlled environments and never against real services without proper authorization.

---

## 🪟 Installation (Windows)

Download the zip file and extract it.

# Install dependencies
# (example for Python)
pip install -r requirements.txt

# Go back to the file and run it.

<h1> FOR THE LINUX </h1>
# Update system
sudo apt update

# Install dependencies (example for Python)
sudo apt install python3 python3-pip -y

# Clone the repository
git clone https://github.com/botucatu1/cyber-phantom-DDOS-
cd cyber-phantom-DDOS-

# Install requirements
pip3 install -r requirements.txt

# Run the project
python3 main.py
